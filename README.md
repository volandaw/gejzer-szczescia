### 📝 O projekcie
Projekt "Gejzer Szczęścia" to interaktywna aplikacja analityczna oparta na danych **World Happiness Report**. 
Łączy ona analizę historyczną z nowoczesnym podejściem do prognozowania przy użyciu Sztucznej Inteligencji.

### 🧠 Technologia
* **Model ML:** Extra Trees Regressor (zoptymalizowany przez bibliotekę PyCaret).
* **Interfejs:** Streamlit.
* **Wizualizacja:** Plotly (interaktywne wykresy liniowe).

### 🚀 Funkcje aplikacji
1. **Analiza historyczna:** Przegląd wskaźników szczęścia dla wybranych krajów od 2006 roku.
2. **Symulacja "What-If":** Możliwość indywidualnego sterowania wzrostem PKB i wsparcia społecznego dla każdego kraju z osobna.
3. **Prognoza AI:** Wykorzystanie wytrenowanego modelu do przewidywania trendów w latach 2026-2028.

### 🛠️ Jak uruchomić lokalnie?
1. Sklonuj repozytorium.
2. Zainstaluj wymagane biblioteki: `pip install -r requirements.txt`.
3. Uruchom aplikację: `streamlit run aplikacja_final.py`.
